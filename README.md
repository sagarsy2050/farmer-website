## How to Deploy on GitHub Pages

1. Create a new repository on GitHub named `farmer-website`
2. Clone the repository to your local machine:
   ```bash
   git clone https://github.com/yourusername/farmer-website.git
   cd farmer-website
   ```
3. Copy all files from this folder to the repository
4. Add all files:
   ```bash
   git add .
   ```
5. Commit your changes:
   ```bash
   git commit -m "Initial commit: Farm products website"
   ```
6. Push to GitHub:
   ```bash
   git push -u origin main
   ```
7. Go to repository settings → Pages → Select "main" branch → Save
8. Your site will be live at: `https://yourusername.github.io/farmer-website/`
